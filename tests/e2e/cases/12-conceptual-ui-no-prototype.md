# 12-conceptual-ui-no-prototype: UI 相关但概念性问题不自动出原型

**规则**（来自 `prompts.ts` "视觉辅助原则"）：
- UI 话题不自动等于视觉问题
- 权限、规则、风险、范围讨论应优先留在文字里

## 前置

- 干净加载

## 步骤

1. 输入：`我想做一个管理员后台，维护司机卡 ID。这个后台里权限应该怎么分？`
2. 等 AI 回复完成
3. 读取 AI 回复文字和右侧预览区状态

## 期望

- AI 回复气泡文字 ≥ 50 字符
- AI 回复应围绕权限/角色/操作范围讨论，含"权限"、"角色"、"管理员"、"操作"中的至少两个词
- AI 不应因为出现"后台"就自动生成完整原型
- 右侧预览区应仍为空/欢迎态，或保持之前状态；本轮不应出现新的完整 iframe 原型内容
- 如果 AI 提到原型，应只是轻量 offer，不应本轮直接画

## 通过标准

AI 能把 UI 相关但概念性的权限问题留在文字里处理，不自动进入视觉原型模式。
